Nutzungsweise der Tags Die Tags helfen den Trainer die Stärken, Schwächen und Verbesserungwünsche seiner Spieler im Blick zu behalten und so gezielter sein Training zu gestalten. Weiterhin werden mit den Tags in der Zukunft verschiedene Bereiche von Volleymotion miteinander verknüpft. Beispielsweise:
  • Artikelvorschläge anhand der Tags
  • Übungsvorschläge anhand der Tags
  • Trainingsvorschläge anhand der Tags
  • Vorschläge von YouTube Videos anhand der Tags
